# LexClare

## Description
- A simple word search web app
- The essence of this project is go gain mastery and understanding, working work with APIs in creating server-side logic

## Tools 
- EJS
- Node JS
- Express JS
- JavaScript
- CSS
- Axios

## Basic Features
- User can simply type in a word or a phrase and the word/phrase and its homophones will be displayed.
- The word will be displayed together with part of speech, short definitions, and phonetic transcription.
- User can also learn how the word is pronounced by pressing the volumne button on the left side of it.
